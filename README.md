# Quiz Pro ローカル開発環境
これは転職活動用に作成されたポートフォリオのローカル開発環境の構築用のリポジトリです。

## セットアップ
(1)APIとFRONTのリポジトリを以下のコマンドでダウンロードします
```
bash scripts/setup.sh
```

(2)docker composeでコンテナを立ち上げます
```
docker compose up --build
```
